# e-mail-marketing
E-mail marketing estruturado em HTML e CSS
Croiação de e-mail marketing com HTML e CSS para campanha de produtos novos no portofólio de empresa. 
Modelo pode ser usado para qualquer campanha, e em qualquer plataforma, pois é adptada para isso. Além de possuir compatilidade com diversas versões do Outlook, G-mail, e responsividade. 

Para conferir, acesse: https://carvalhomatheuss.github.io/e-mail-marketing/
